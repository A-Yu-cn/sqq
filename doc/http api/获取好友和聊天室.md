##### 简要描述

- 获取好友和聊天列表

##### 请求URL

- ` /users/friends_and_chatroom `

  > 需要在headers中添加`Authorization`字段

##### 请求方式

- GET

##### 参数

- 无

##### 返回示例 

```json
{
    "mes": "",
    "data": {
        "friends": [
            [
                12345,
                "测试"
            ]
        ],
        "chatroom_list": [
            [
                123456,
                "测试"
            ],
            [
                982625,
                "巨无霸聊天室"
            ],
            [
                999564,
                "巨无霸聊天室"
            ]
        ]
    }
}
```
