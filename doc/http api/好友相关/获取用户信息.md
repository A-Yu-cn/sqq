##### 简要描述

- 获取用户信息

##### 请求URL

- ` /users/<user_id> `

  > 例：`/users/12345`

##### 请求方式

- GET 

##### 参数



| 参数名  | 必选 | 类型 | 说明             |
| :------ | :--- | :--- | ---------------- |
| user_id | 是   | int  | 路径参数，用户id |



##### 返回示例 

```
{
    "mes": "",
    "data": {
        "id": 22275,
        "nickname": "楷禅",
        "email": "179167092@qq.com"
    }
}
```
