##### 简要描述

- 查询用户或聊天室

##### 请求URL

- ` /query/<query_id>`

##### 请求方式

- GET

##### 参数



| 参数名   | 必选 | 类型 | 说明                         |
| :------- | :--- | :--- | ---------------------------- |
| query_id | 是   | int  | 用户id或聊天室, **路径参数** |

##### 返回示例 

```json
{
    "mes": "",
    "data": {
        "id": "96568",
        "name": "楷禅",
        "email": "123456@qq.com", // 查询为结果为用户时才有
    }
}
```
