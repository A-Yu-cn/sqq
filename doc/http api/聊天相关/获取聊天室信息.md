##### 简要描述

- 获取聊天室信息

##### 请求URL

- ` /chatroom/<room_id>`

##### 请求方式

- GET

##### 参数



| 参数名  | 必选 | 类型 | 说明               |
| :------ | :--- | :--- | ------------------ |
| room_id | 是   | int  | 聊天室id，路径参数 |
