##### 简要描述

- 文件查询

##### 请求URL

- ` /file/query`

##### 请求方式

- POST

  > 需要在headers中添加Authorization字段

##### 参数

| 参数名   | 必选 | 类型   | 说明      |
| :------- | :--- | :----- | --------- |
| file_md5 | 是   | string | 文件MD5值 |



##### 返回示例 

```json
{
    "mes": "",
    "data": "static/file_upload/a4db6eaab777e114689ea92f40c176a7.html"
}
```



- 查询失败时mes字段为N
