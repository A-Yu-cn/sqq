##### 简要描述

- 上传文件

##### 请求URL

- ` /file/`

##### 请求方式

- POST

  > 需要在headers中添加Authorization字段

##### 参数

| 参数名 | 必选 | 类型 | 说明 |
| :----- | :--- | :--- | ---- |
| file   | 是   | file | 文件 |

- python 文件上传示例

```
files = {
    'file': ('filename.jpg', open('filename.jpg', 'rb'))
}
response = requests.post('http://127.0.0.1:11111/file/', files=files, headers={
    'Authorization': '3'
})
```





##### 返回示例 

```json
{
    "mes": "",
    "data": {}
}
```
