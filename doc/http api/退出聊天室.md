##### 简要描述

- 退出聊天室

##### 请求URL

- ` /chatroom/ `

  > 需要在headers中添加Authorization字段

##### 请求方式

- DELETE

##### 参数



| 参数名      | 必选 | 类型 | 说明     |
| :---------- | :--- | :--- | -------- |
| chatroom_id | 是   | int  | 聊天室id |

##### 返回示例 

```json
{
    "mes": "",
    "data": {}
}
```
