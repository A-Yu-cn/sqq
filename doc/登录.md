##### 简要描述

- 登录

##### 请求URL

- ` /users/auth `

##### 请求方式

- POST 

##### 参数



| 参数名   | 必选 | 类型   | 说明         |
| :------- | :--- | :----- | ------------ |
| identity | 是   | string | 账号或者邮箱 |
| password | 是   | string | 密码         |
